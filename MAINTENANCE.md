# 持续维护规则

本项目由 Gardner's Vault 持续维护。维护不承诺固定更新频率，以下情形将触发评估：

- 与未上市公司 PE/VC 股权融资相关的重要法律规则或监管口径发生变化；
- 市场实践、核心条款采用方向或谈判尺度出现值得纳入的变化；
- WorkBuddy、SkillHub、Codex 或相关文件处理能力发生兼容性变化；
- 用户反馈、测试或真实使用发现可复现的错误、遗漏或不稳定行为。

每次正式发布至少完成：版本号和复核日期更新、变更记录、源项目测试、公开发布测试、隐私扫描、一致性校验、发布清单和 SHA-256 校验。高风险法律结论仍应核验最新一手依据。

一般问题请通过 GitHub Issues 提交；不要在公开 Issue 中上传交易文件、个人信息或其他保密内容。公众号：加德纳的宝匣。
