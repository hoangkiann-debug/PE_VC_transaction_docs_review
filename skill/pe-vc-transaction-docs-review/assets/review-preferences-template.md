# Review Preference Profile

这是一份可选的长期审阅偏好表。只填写可以跨项目复用的通用经验，不填写客户名称、项目名称、交易对方、具体项目金额或其他保密信息。

## 基本偏好

- 偏好表名称：
- 适用范围：人民币境内 / 境外美元直持 / VIE / 全部
- 常用审阅立场：公司及创始方 / 投资人 / 领投 / 跟投 / 战略投资人
- 风险尺度：审慎 / 平衡 / 进取 / 逐项说明
- 默认交付：完整报告 / Word批注 / 红线稿 / Major Issue List / 组合
- 输出风格：简洁 / 详细 / 仅列重大问题 / 其他

## 工作方式

- 是否默认检查跨文件冲突：是 / 否
- 是否默认建立 Major Issue List：是 / 否
- 市场数据呈现：整数级大约比例 / 仅说明主流方向 / 明确要求时提供精确值
- 法律问题提示深度：简要 / 详细
- 其他长期偏好：

## 更新规则

- 只有在用户明确确认“以后都按这个标准”时，才更新本偏好表。
- 当前项目的特殊事实、金额、对方立场或一次性让步，不自动写入本偏好表。
- 本次明确指令和当前项目事实始终优先于本偏好表。
