## 3. Detailed Review Comment

### PEVC-001 创始人个人回购责任

- 文件及位置：股东协议第10.2条
- 当前原文：发生任何回购事件时，公司及创始人应连带向投资人支付全部回购价款。
- 问题及风险：任何回购事件均触发创始人无限连带责任，未区分公司原因、投资人原因、客观上市失败和创始人过错，也未设置责任顺位或上限。
- 我方立场：公司承担第一顺位义务；创始人仅在欺诈、故意违约或其个人可控制的特定触发事件下承担补充责任。
- 市场背景：回购义务主体、创始人责任顺位和责任上限存在多种做法。实际交付时应从 `references/benchmark-data.json` 调取与架构、年份和条款口径匹配的数据，并明确其只是历史谈判背景。
- 法律核验：需结合现行公司法、资本维持和减资程序核验公司履行路径。涉及中国法律相关事项，请咨询执业中国律师；涉及非中国法管辖事项，请咨询相应法域的执业律师。
- 首选修改：

> 公司应在适用法律允许的范围内承担回购义务。仅因创始人存在欺诈、故意违约或其个人可合理控制且直接导致回购事件发生的行为，且公司未能在约定期限内足额支付回购价款时，相关创始人方就未支付部分承担补充责任；创始人方的累计责任以其届时直接或间接持有的公司股权公允价值为上限，但欺诈或故意侵占公司、投资人财产的除外。

- 平衡方案：创始人承担第二顺位责任，责任上限为其持股价值与已取得税后收益之和。
- Fallback：至少删除一般违约和客观上市失败下的创始人个人责任，并明确公司先履行、创始人后履行。
- 待确认事实：投资人是否接受公司先履行；创始人是否提供其他担保；回购价格是否与清算优先、违约金重复计算。

## 4. Cross-document Conflict

### PEVC-006 董事会表决门槛不一致

- 股东协议：5名董事中至少4名同意，且须包含投资人董事同意。
- 公司章程：5名董事中过半数同意。
- 风险：同一事项可能在合同层面和公司治理层面产生不同结果，影响决议效力、违约判断和登记文件一致性。
- 建议：根据最终谈判结果统一董事人数、提名权、法定人数、普通事项门槛和保留事项门槛，并在签署前复核两份文件的定义和附件。
